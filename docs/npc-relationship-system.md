# NPC Relationship System

Relationship levels are normalized in `npc_relationship_levels`.

Supported levels:

- Stranger
- Acquaintance
- Trusted
- Friend
- Close Friend
- Family
- Mentor
- Rival
- Companion
- Soul Bond

Relationship data can modify prices, dialogue, quests, gifts, Nilalang trades, equipment crafting, and skill teaching. Phase G stores points and level references; future save data should track per-player relationship state separately from canonical NPC templates.
