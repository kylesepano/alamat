# NPC Asset Guide

Every NPC has asset prompt data in `npc_asset_prompts`.

Required asset concepts:

- Portrait
- Overworld sprite
- Battle sprite when recruitable
- Bust illustration
- Emotion sheet
- Voice description

Asset directories:

- `public/assets/npcs/portraits`
- `public/assets/npcs/sprites`
- `public/assets/npcs/busts`
- `public/assets/npcs/emotions`
- `public/assets/npcs/voices`
- `public/assets/npcs/generated`

Prompt rules:

- Filipino fantasy RPG style
- Respect regional traditions
- No text in visual assets
- Strong silhouette for sprites
- Profession tools should be visible
- Avoid reducing living traditions to costume decoration
