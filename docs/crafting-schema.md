# Crafting Schema

Primary crafting tables:

- `crafting_stations`
- `recipe_categories`
- `recipes`
- `recipe_ingredients`

Recipes support cooking, herbalism, forging, weaving, carving, alchemy, ritual crafting, enchanting, equipment upgrades, talisman making, monster care, fishing, farming, and festival crafting.

Recipe outputs can reference either `items` or `equipment`.
