# World Asset Guide

Every location has:

- `map_filename`
- `preview_image`
- `asset_prompt`

Asset folders:

- `public/assets/world/maps`
- `public/assets/world/previews`
- `public/assets/world/tilesets`
- `public/assets/world/markers`
- `public/assets/world/generated`

Map prompt rules:

- Classic handheld top-down Filipino fantasy RPG
- Readable tile-based layout
- No text
- No UI
- Clear paths
- Clear collision objects
- Distinct landmarks
- Practical for Tiled or React implementation
