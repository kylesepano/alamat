# Story Asset Guide

Story asset prompts are stored in `story_asset_prompts`.

Supported asset types:

- cutscene illustrations
- dialogue portraits
- chapter banners
- act key art
- storybook pages
- historical murals
- ancient maps
- memory sequences
- festival artwork
- ending illustrations

Every prompt should include the intended owner record, filename, visual goal, negative prompt, and style notes. The visual language should remain Filipino fantasy, grounded, respectful, and readable inside game UI.
