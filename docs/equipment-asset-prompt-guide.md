# Equipment Asset Prompt Guide

Every equipment record must include:

- `icon_filename`
- `sprite_filename`
- `thumbnail_filename`
- `asset_prompt`
- `icon_prompt`
- `design_notes`

Asset directories:

- `public/assets/equipment/icons`
- `public/assets/equipment/sprites`
- `public/assets/equipment/thumbnails`
- `public/assets/equipment/placeholders`
- `public/assets/equipment/generated`

Prompt style:

- Transparent background
- No text
- No UI frame
- Readable at 32x32 and 64x64
- Strong silhouette
- Filipino fantasy RPG aesthetic
- Respectful use of woven patterns, bamboo, brass, shell, rattan, river stone, leaf fiber, and ceremonial cloth

Equipment prompts should not be reused for consumable items. Equipment should read as worn, wielded, equipped, or set-linked objects.
