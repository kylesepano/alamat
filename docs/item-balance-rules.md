# Item Balance Rules

Phase E item balance is conservative because full combat is not implemented yet.

Rules:

- Common consumables should provide small flat recovery.
- Food should favor field recovery and trust scenes.
- Herbs should support remedies and crafting.
- Offerings should affect trust, affinity, habitat, weather, or specific Nilalang links.
- Quest and key items should not be sellable.
- Crafting materials must be usable in crafting.
- Monster drops should be respectful encounter traces, not trophy framing.
- Skill materials should reference skill systems by `skill_id` but not duplicate skill definitions.

Future combat phases can tune `effect_payload` values against the Phase D Combat Bible formulas.
