# Localization Guide

Phase K prepares story data for localization through `locale_key` fields on dialogue nodes and authoring notes on folk tales.

Guidelines:

- Keep regional voices respectful.
- Avoid caricatured dialect.
- Preserve uncertainty when multiple in-world traditions disagree.
- Translate tone and intention, not only literal words.
- Keep gameplay values such as Mercy, Justice, Curiosity, Wisdom, Duty, Compassion, Honor, Tradition, Balance, and Sacrifice consistent across UI and story records.

The `LocalizationService` can expose current dialogue keys for export tooling.
