# Endings

Ending data is stored in `ending_routes` and `epilogues`.

Supported route families:

- True Ending
- Regional Endings
- Faction Endings
- Companion Endings
- Legendary Ending
- Hidden Ending
- Postgame Ending
- Expansion Ending

Ending requirements are JSON payloads so future systems can evaluate values, flags, relationship state, faction reputation, quest routes, Nilalang trust, and world state without schema changes.
